# 學習資源

歡迎來到 **CertGym 證照道館**的學習資源中心！

我們精心整理了各大熱門證照的備考資源——閃卡素材、題庫解析、考試技巧、學習路線圖與社群心得。
每一份資源，都是為了讓你的備考更有方向、更有效率，更快通過心目中的夢想證照。

## 資源分類

- **閃卡素材**：依科目分類的精選閃卡，搭配間隔重複法高效記憶考點
- **題庫解析**：熱門證照模擬題與詳細解析，熟悉出題邏輯與答題技巧
- **備考路線圖**：從零開始到考試上榜，一步步規劃你的備考時間表
- **考試攻略**：各大證照的報名流程、考試格式、重點章節與雷區提醒
- **錯題筆記**：常見易錯題型彙整，精準補強知識盲點
- **社群心得**：考生分享的實戰備考故事、通過心得與學習方法

## 如何開始備考？

1. 從左側選單選擇你目標的證照類別
2. 瀏覽閃卡素材與題庫解析，建立紮實基礎
3. 下載 CertGym App，透過閃卡、題庫與錯題收藏高效練習
4. 加入學習社群，與考生互助交流，持續維持備考動力
